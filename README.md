# diogo.a.gulhak.github.io

Portfolio pessoal em `Next.js`, `Tailwind CSS` e `shadcn/ui`, com export estatico para GitHub Pages.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui

## Scripts

- `npm run dev`: ambiente local
- `npm run lint`: validacao do codigo
- `npm run build`: build de producao com export estatico

## Estrutura

O projeto ativo esta concentrado em:

- `src/app`: App Router e estilos globais
- `src/components`: componentes de interface e secoes do portfolio
- `src/data`: conteudo estruturado do site
- `public/images`: assets usados pela versao atual
- `.github/workflows/pages.yml`: deploy para GitHub Pages

Documentacao complementar da arquitetura atual: `docs/portfolio-architecture.md`.
